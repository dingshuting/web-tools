# web-tools
Builded with spring-boot, that include CURD operations on MySql and MongoDB and show them on the Web by HLoader or Manager in WebApp folder.
**项目为初始化版本，如果有什么不足、bug或其它想法可留言讨论**
## web-tools 提供一个快递搭建一个后台数据管理的系统

整个系统为前后端分析方式，后端采用spring-boot作为启动项目，springMVC、springSecurity、HibernateJPA作为后台主要的框架组件；
管理端页面采用HTML/CSS,JS采用[javaScriptMVC](http://www.javascriptmvc.com)为基础框架
后台项目通过启动bootstrap项目来启动后台主项目，其依赖core,mongo_support为可选依赖
