-------------V1.0--------------------
本包提供了MongoDB的支持，同时也通过在mysql中的配置确认在MongoDB中的数据格式和关系。
整个包提供了业务的动作扩展功能，业务的逻辑关系在ExtraData和ExtraDataCols 2个数据表中完成配置。
ExtraData- 扩展的数据表的描述
ExtraDataCols- 扩展表的明细字段的定义，包括字段类型和验证规则等。
整个业务包流程为
	1,配置业务数据表的结构 ----↓  
	2,页面或客户端提交相应的json数据到指定的control，control同时对集合名称进行相应的匿名化，防止数据表名称保留在前台影响数据安全-↓
	3,control对页面数据进行收集，和类型的转化后调用service的相应CURD方法-↓
	4,service层调用相关的监听（ServiceListener的实现类，通过注册器注册即可实现服务层调用），实现数据的合法性、有效性以及其它扩展的功能-↓
	5,数据落地
	6,进行数据的查询统计及其它操作
